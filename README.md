# Customer Shopping Behavior Analysis

## 📌 Project Overview
This project analyzes 3,900+ customer transactions to identify spending patterns, customer segments, and product preferences.

## 🔧 Tools Used
- Python (Pandas)
- SQL (PostgreSQL)
- Power BI
- Excel

## 📊 Key Insights
- Male customers generate ~2x more revenue
- Young adults contribute highest revenue (~$62K)
- ~80% customers are loyal
- Discounts do not always reduce spending
